# License Wizard
A visual, interactive license generator that helps you choose (or build) the exact license you want.

This tool supports: <br>
✔ Open-source licenses (MIT, BSD, Apache, GPL, AGPL, LGPL, MPL)  
✔ Closed-source restrictions  
✔ Non-commercial licenses  
✔ Network/SaaS copyleft detection  
✔ “Strict clauses” (Anti-AI, No-Gov, No-Big-Tech, Moral Use, etc.)  
✔ Custom Griffin-style licenses  
✔ Export to text or PDF  
✔ GitHub Action auto-injector for LICENSE file

## Deploying the Wizard
You can host the wizard for free using GitHub Pages.

**Enable Pages:**

1. Go to **Settings → Pages**
2. Set **Branch:** `main`  
   **Folder:** `/root`
3. Save.

Your wizard will be live at:
https://ginesthoii.github.io/REPO/

---

## Embedding the Wizard in Any Website

Copy/paste this anywhere:

```html
<iframe 
    src="https://ginesthoii/your-wizard/index.html" 
    style="width:100%; height:900px; border:none; border-radius:8px;">
</iframe>
```
Security Note:
The wizard is static and safe — no cookies, no sessions, no tracking.
